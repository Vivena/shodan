Lit un block dans le disque.

Il s'agit de la lecture des données dans le fichier du disque en utilisant la fonction read_physical_block. ici, on prend en compte les données mises en place dans le cache avant de lire.

Parameters:
idL'identifiant du disque.
blockLe block (contenant les 1024 octets) à lire dans le disque.
numLe numéro du block à lire dans le disque.
Returns:
Un error, à valeur -1 s'il y a un problème avec la fonction C read.
